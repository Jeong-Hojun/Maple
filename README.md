# 진의 신비한 정원 도우미

의존성 없이 바로 열 수 있는 정적 웹앱입니다.

## 기능

- 클릭, 드래그 앤 드롭, 붙여넣기 방식의 이벤트 스크린샷 업로드
- **Gemini Vision AI** 기반 보드·주사위·진 위치 자동 탐지
- 탐지 결과 검토 및 부분 수정 (11×11 보드 그리드 UI)
- 모든 주사위 순서 기대 비료 비교
- 물음표 발판 좌/우 기대값 비교
- 몬스터/이동 효과 반영

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. [Google AI Studio](https://aistudio.google.com/app/apikey)에서 Gemini API 키를 발급받아 입력합니다.
3. 이벤트 캡처 화면을 업로드합니다.
4. 자동 탐지된 보드 상태와 주사위를 검토하고 필요한 값만 수정합니다.
5. 기대 비료가 가장 높은 순서를 확인합니다.

## 탐지 방식

- Gemini Vision AI가 이미지에서 보드 40개 발판의 비료값, 발판 종류, 주사위 3개 눈금, 진 캐릭터 위치를 자동 분석합니다.
- API 키는 브라우저 `localStorage`에만 저장되며 서버로 전송되지 않습니다.
- 탐지 결과가 틀린 경우 보드 그리드에서 직접 수정할 수 있습니다.

## 보드 구조

총 40칸, 시계 반대 방향으로 순환합니다.

| 구간 | 인덱스 | 설명 |
|------|--------|------|
| 하단 행 | 0~10 | 우하단(START) → 좌하단(+10칸이동) |
| 왼쪽 열 | 11~19 | 아래 → 위 |
| 상단 행 | 20~30 | 좌상단 → 우상단(?) |
| 오른쪽 열 | 31~39 | 위 → 아래 |

고정 랜드마크: 인덱스 0 = START, 인덱스 10 = +10칸이동, 인덱스 30 = ?

## 로컬 서버 실행 (선택)

Perl이 설치된 경우:

```
perl serve.pl
```

`http://localhost:3000` 에서 접속합니다.
