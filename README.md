# 진의 신비한 정원 도우미 초안

의존성 없이 바로 열 수 있는 정적 웹앱 초안입니다.

## 포함한 범위

- 클릭, 드래그 앤 드롭, 붙여넣기 방식의 이벤트 스크린샷 업로드
- 업로드 이미지 기준 보드와 주사위 자동 탐지
- 탐지 결과 검토 및 부분 수정
- 모든 주사위 순서와 특수효과 사용 케이스 완전탐색
- 물음표 발판의 좌/우 기대값 비교
- 몬스터/발판 효과를 데이터 기반으로 반영

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 이벤트 캡처 화면을 업로드합니다.
3. 자동 탐지된 보드 상태와 주사위를 검토하고 필요한 값만 수정합니다.
4. 기대 비료가 가장 높은 순서를 확인합니다.

## 현재 가정

- 현재 자동 탐지는 템플릿 기반 보드 위치 추정 + OCR + 색상 분류 조합입니다.
- 질문 발판은 기본적으로 좌/우 중 더 높은 기대값을 주는 경로를 선택합니다.
- 특수 주사위와 몬스터 효과는 편집 가능한 데이터 형태로 두어, 실제 규칙이 확정되면 UI 옵션만 세밀하게 맞춰 확장할 수 있습니다.

- 실시간 화면 공유 입력으로 전환
